# Workspace Agent Rules

These rules govern agent behavior within the `Hayday-Bot-Pro` workspace.

## Branch Protection

- **DO NOT** push code directly to the `main` branch.
- Always use feature branches (e.g., `feature/xyz` or `fix/abc`) for development.
- Commits must be verified and clean before being merged.
- Only merge changes to `main` via pull requests or after explicit user review and confirmation.
